# Ship_Fault_Diagnosis_QA_System
Ship Fault Diagnosis QA Assistant: an offline-first Graph-RAG system for ship machinery fault diagnosis, combining hybrid retrieval, knowledge graph reasoning, and evidence-grounded answer generation with expert review.  面向船舶机电故障诊断的离线优先 Graph-RAG 问答系统，融合混合检索、知识图谱推理与可追溯证据生成，并支持专家审图闭环审核。
